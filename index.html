Major Projects

  • Yelp Restaurant Reviews Aspect based Rating using Latent Dirichlet Allocation

    Performed sentiment analysis and predicted top positive and top negative reviews for restaurants. Calculated aspect (food, service, ambiance) based rating for each review using Latent Dirichlet Allocation by employing guided seed words and unsupervised pruning to significantly reduce the convergence time and improving accuracy of the clustering by 20%. Computed overall aspect ratings of each restaurant by combining the individual aspect ratings of each review with probabilities generated by LDA algorithm. Recommended top 5 dishes for a restaurant using the most frequent dishes mentioned in the reviews

  • Car Insurance Fraud Detection

    Performed exhaustive feature engineering, mitigated class imbalance using SMOTE and one hot encoded categorical features. Classified data using Logistic Regression, KNN, Decision Trees, Random Forests and Linear Discriminant Analysis. Evaluated the performance using Cohen’s Kappa, Area Under the Curve metrics and ROC curves.

  • Action Recognition using LSTMs and SVM in Pytorch

    Employed transfer learning to extract image features from a pre-trained VGG network with 16 layers and built an LSTM with these features for predicting the action being done in a video. Built a competing SVM model and analyzed performance of both models.
